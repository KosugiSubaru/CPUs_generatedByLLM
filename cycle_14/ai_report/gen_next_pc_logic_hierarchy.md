教育用シングルサイクルCPUにおいて、**next_pc_logic**（ネクストPCロジック）は、プログラムの流れを決定する「分岐・ジャンプの判定」と「ターゲットアドレスの計算」を担います。

ISAで定義された「PC+2」「PC+imm」「rs1+imm」という3つの遷移先を、演算フラグと命令種別に基づいて切り替える構造を、以下の3階層で提案します。

---

### 1. 階層構造の概要

| 階層レベル | モジュール名 | 役割 | パタン構造化の適用 |
| :--- | :--- | :--- | :--- |
| **L0 (ユニット)** | `next_pc_logic` | 最上位モジュール。ターゲット計算、条件判定、遷移先選択を統合。 | 各ブロックの接続 |
| **L1 (機能ブロック)** | `next_pc_logic_adder_16bit` | 16bit加算器。PC+immやrs1+immの計算に使用。 | `generate`によるフルアダーの連結 |
| | `next_pc_logic_mux3_16bit` | 16bit幅の3入力セレクタ。次PCの最終候補を選択。 | `generate`による1bitセレクタの並列化 |
| | `next_pc_logic_branch_eval` | 分岐条件（N^VやZ）を判定し、分岐実行の有無を決定。 | 論理ゲートによる判定回路 |
| **L2 (プリミティブ)** | `next_pc_logic_full_adder` | 1ビット全加算器。 | 基本論理ゲート |
| | `next_pc_logic_mux3_1bit` | 1ビット幅の3入力セレクタ。 | 基本論理ゲート |

---

### 2. 各ファイルの役割と階層レベルの説明

#### 【L2: プリミティブ層】
最小単位の論理を定義します。回路図上で「基本パーツ」として視覚化されます。

1.  **next_pc_logic_full_adder.v**
    *   **役割:** 1ビットの加算を行います。
2.  **next_pc_logic_mux3_1bit.v**
    *   **役割:** 3つの入力（PC+2、PC+imm、rs1+immの各1ビット分）から、制御信号に応じて1つを選択します。

#### 【L1: 機能ブロック層】
プリミティブを並列化・連結し、ISAの仕様を具体化します。

3.  **next_pc_logic_adder_16bit.v**
    *   **役割:** ターゲットアドレス計算用の加算器。
    *   **構造:** `generate` 文を用いて `next_pc_logic_full_adder` を16個インスタンス化し、リップルキャリー構造を作ります。
4.  **next_pc_logic_mux3_16bit.v**
    *   **役割:** 16ビット幅の次PC選択器。
    *   **構造:** `generate` 文を用いて `next_pc_logic_mux3_1bit` を16個並列配置します。
    *   **視覚化の利点:** ビットごとに並列にデータが流れる様子が視覚化されます。
5.  **next_pc_logic_branch_eval.v**
    *   **役割:** ISAの「behavior」欄にある条件（例：blt命令なら `N^V`）を評価し、実際に分岐するかどうかを判定します。

#### 【L0: ユニット層（最上位）】
CPUデータパス全体とのインターフェースです。

6.  **next_pc_logic.v**
    *   **役割:** 以下の3つのパスを統合し、最終的な `next_pc` アドレスを `pc_reg` へ出力します。
        1.  **通常パス:** `PC + 2`
        2.  **分岐・JALパス:** `PC + imm` (計算ブロック)
        3.  **JALRパス:** `rs1 + imm` (計算ブロック)
    *   **視覚化の利点:** 分岐成立時と不成立時で、どの加算器の結果が選ばれるのかが結線として明確に示されます。

---

### 3. 提案するサブモジュールリスト

submodules:
next_pc_logic: 次のPCアドレスを決定する最上位ユニット。ターゲットアドレス計算と選択論理を統合。
next_pc_logic_adder_16bit: 16ビットの加算器。ターゲットアドレス（PC+imm, rs1+imm）の計算に使用。
next_pc_logic_mux3_16bit: 16ビット幅の3入力セレクタ。次PCの最終候補を1つに絞り込む。
next_pc_logic_branch_eval: 分岐・ジャンプの成立条件を評価する論理ユニット。フラグと命令を照合する。
next_pc_logic_full_adder: 加算器を構成する最小単位の1ビット全加算器。
next_pc_logic_mux3_1bit: セレクタを構成する最小単位の1ビット3入力セレクタ。
