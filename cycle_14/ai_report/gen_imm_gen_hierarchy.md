教育用シングルサイクルCPUにおいて、**imm_gen**（イミディエイト・ジェネレータ）は、命令語の異なる位置に埋め込まれた即値（イミディエイト）を抽出し、データ幅を16ビットに整える（符号拡張する）役割を担います。

ISAの定義に基づき、即値の抽出パターンごとに独立したブロックを設け、それらをセレクタで切り替える階層構造を提案します。

---

### 1. 階層構造の概要

| 階層レベル | モジュール名 | 役割 | パタン構造化の適用 |
| :--- | :--- | :--- | :--- |
| **L0 (ユニット)** | `imm_gen` | 最上位。抽出器とセレクタを統合し、命令に応じた即値を出力。 | 構成要素の接続 |
| **L1 (機能ブロック)** | `imm_gen_ext4_i` | 命令[7:4]を抽出し符号拡張。(`addi`, `load`, `jalr`用) | 符号ビットの複製(Wire) |
| | `imm_gen_ext4_s` | 命令[15:12]を抽出し符号拡張。(`store`用) | 符号ビットの複製(Wire) |
| | `imm_gen_ext8` | 命令[11:4]を抽出し符号拡張。(`loadi`, `jal`用) | 符号ビットの複製(Wire) |
| | `imm_gen_ext12` | 命令[15:4]を抽出し符号拡張。(`blt`, `bz`用) | 符号ビットの複製(Wire) |
| | `imm_gen_mux5_16bit` | 5入力16bitセレクタ。適切な抽出結果を選択。 | `generate`による1bit幅セレクタの並列化 |
| **L2 (プリミティブ)** | `imm_gen_mux5_1bit` | 5入力1bitセレクタ。MUXの最小単位。 | 基本論理ゲート |

---

### 2. 各ファイルの役割と階層レベルの説明

#### 【L2: プリミティブ層】
最小の論理単位です。

1.  **imm_gen_mux5_1bit.v**
    *   **役割:** 5つの入力（各抽出器の結果）から1つを選択する1ビットセレクタ。制御ユニットからの選択信号に基づきます。

#### 【L1: 機能ブロック層】
ISAの「ビットフィールド」定義に直接対応する抽出ロジックです。

2.  **imm_gen_ext4_i.v** / **imm_gen_ext4_s.v** / **imm_gen_ext8.v** / **imm_gen_ext12.v**
    *   **役割:** 命令語（16bit）から特定のビット範囲を抜き出し、最上位ビット（符号ビット）を16ビット目までコピーして埋めます。
    *   **視覚化の利点:** 回路図上で「どの命令フィールドが即値として使われているか」が、配線の束として明確に示されます。
3.  **imm_gen_mux5_16bit.v**
    *   **役割:** 16ビット幅の5入力セレクタ。
    *   **構造:** `generate` 文を用いて `imm_gen_mux5_1bit` を16個並列にインスタンス化します。
    *   **視覚化の利点:** パタン構造化により、ビット並列処理の構造が美しく視覚化されます。

#### 【L0: ユニット層（最上位）】
各抽出器とセレクタを結合します。

4.  **imm_gen.v**
    *   **役割:** 命令語（`i_instr`）と制御信号を受け取り、最終的な16bit即値を出力します。
    *   **内部構成:** すべての抽出器（L1）を同時に動かし、その出力を `imm_gen_mux5_16bit` で1つに絞り込みます。

---

### 3. 提案するサブモジュールリスト

submodules:
imm_gen: 即値生成ユニットの最上位。抽出器と選択ロジックを統合する。
imm_gen_ext4_i: I形式（addi, load, jalr）の4bit即値を抽出し、16bitに符号拡張する。
imm_gen_ext4_s: S形式（store）の4bit即値を抽出し、16bitに符号拡張する。
imm_gen_ext8: L形式（loadi, jal）の8bit即値を抽出し、16bitに符号拡張する。
imm_gen_ext12: B形式（blt, bz）の12bit即値を抽出し、16bitに符号拡張する。
imm_gen_mux5_16bit: 5つの抽出結果（各形式＋デフォルト）から1つを選択する16bitセレクタ。
imm_gen_mux5_1bit: 1bit幅の5入力セレクタ。16bitセレクタの基本単位。
