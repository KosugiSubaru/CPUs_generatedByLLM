提供されたISA定義とシステムアーキテクチャに基づき、教育用シングルサイクルCPUを構成する1階層下のサブモジュールを提案します。

この構成は、データパスの流れ（命令の取り込みから実行、書き戻しまで）を視覚的に理解しやすいように分割しています。

---

### 1. サブモジュールの提案

| モジュール名 | 役割・機能説明 |
| :--- | :--- |
| **PC (Program Counter)** | 次に実行する命令のアドレスを保持するレジスタ。クロック同期で更新される。 |
| **Control Unit** | 命令の下位4ビット（opcode）をデコードし、各部（ALU、RegFile、セレクタ等）への制御信号を生成する「司令塔」。 |
| **Register File** | 16個の16bitレジスタを保持。2つのレジスタ読み出し（rs1, rs2）と1つの書き込み（rd）を同時に行う。レジスタ0は常に0を出力する。 |
| **ALU (Arithmetic Logic Unit)** | 加減算、論理演算、シフト演算を実行。演算結果とともに、フラグ（Z, N, V）を計算・出力する。 |
| **ImmGen (Immediate Generator)** | 命令の種類に応じて、特定のビットフィールドを抽出・符号拡張し、16bitの即値を生成する。 |
| **Flag Register** | ALUが出力したフラグ（Z, N, V）を1クロック保持する。条件分岐命令（blt, bz）はこのレジスタの値を見て分岐を判断する。 |
| **PC Control / Adder** | 次のPC値を計算する。通常は `PC+2` だが、分岐・ジャンプ命令時は `PC+imm` や `rs1+imm` を選択する。 |
| **Data Selector (Mux)** | ALUの入力（rs2か即値か）や、レジスタへの書き込みデータ（ALU出力、メモリ出力、PC+2のいずれか）を切り替えるマルチプレクサ群。 |

---

### 2. CPUの動作説明

シングルサイクルCPUとして、1クロックの間に以下のステップが連鎖的に（組合せ回路として）実行されます。

1.  **Instruction Fetch (命令取り出し)**
    *   `PC` が指すアドレスの命令を命令メモリ（imem）から読み出し、`i_instr` として取得します。
2.  **Decode (デコード)**
    *   `Control Unit` が `opcode` を読み取り、演算の種類やメモリ書き込みの有無、セレクタの切り替え信号を即座に生成します。
    *   同時に `ImmGen` が命令形式に合わせて即値を生成します。
3.  **Execute (実行)**
    *   `Register File` から `rs1`, `rs2` のデータが読み出されます。
    *   `ALU` がそれらのデータ（または即値）を用いて演算を行い、結果とフラグを出力します。
    *   **フラグの更新:** ALUが出したフラグは、次のクロックの立ち上がりで `Flag Register` に保存されます（ISAの「1クロック前の参照」制約のため）。
4.  **Memory (メモリ操作)**
    *   `load` 命令の場合は `ALU出力（アドレス）` を用いてデータメモリ（dmem）から読み出し、`store` 命令の場合は `rs2` のデータを書き込みます。
5.  **Write Back (書き戻し)**
    *   `ALUの結果`、`メモリからの読み出しデータ`、または `PC+2（ジャンプ時）` のいずれかを、`rd` レジスタに書き込みます（クロックの立ち上がりで確定）。
6.  **Next PC Calculation (次アドレス計算)**
    *   `Flag Register` の状態と命令の種類に基づき、次の `PC` を決定します。
    *   条件を満たせば `PC + imm`（分岐）、そうでなければ `PC + 2`（通常）となります。

### 教育的ポイント：この構成で視覚化できること
*   **信号の色の変化:** 命令の種類によって、データが流れているパス（有効なセレクタ）と流れていないパスを色分けして表示するのに適しています。
*   **フラグの遅延:** 「今の命令の計算結果」が「次の命令の条件分岐」にどう影響するか、Flag Registerを介した1サイクルのズレを明示的に示せます。
*   **即値の生成:** 命令フォーマット（R型、I型、J型など）によって、ビットの切り出し位置がどう変わるか（ImmGenの動き）を学習できます。
