教育用シングルサイクルCPUにおいて、即値生成器（Immediate Generator）は「命令（バイナリ）の中のどこに数値が埋め込まれているか」を視覚化する重要なモジュールです。

本ISAでは、命令形式によって即値のビット位置と長さ（4bit, 8bit, 12bit）が異なります。これらを「フォーマットごとの抽出・拡張」と「最終的な選択」に階層化することで、ISAのビットフィールド定義と回路の対応を明確にします。

### 1. 階層構造（Hierarchy）の定義

回路図で「どの命令形式の時に、どのビットが抽出され、どう符号拡張されるか」がブロックとして見えるように構成します。

*   **Level 0 (Primitive):** 最小単位（1bit 2-to-1 MUX）。
*   **Level 1 (Extenders):** 各命令形式に応じたビット抽出と符号拡張（16bit化）。
    *   Type A (4bit低位): `addi`, `load`, `jalr` 用
    *   Type B (4bit高位): `store` 用
    *   Type C (8bit): `loadi`, `jal` 用
    *   Type D (12bit): `blt`, `bz` 用
*   **Level 2 (Structural MUX):** Level 1の結果から1つを選択する16bitセレクタ。
*   **Level 3 (Top):** これらを統合する最上位層。

### 2. ファイル構成と役割

1ファイル1モジュールの原則に基づき、以下の7つのファイルで構成します。

| ファイル名（モジュール名） | 役割説明 | 階層レベル |
| :--- | :--- | :--- |
| `imm_generator_mux2.v` | 1bit 2入力セレクタ。16bitセレクタの基礎パーツ。 | Level 0 |
| `imm_generator_ext4_low.v` | `instr[7:4]` を抽出し、16bitに符号拡張する。 | Level 1 |
| `imm_generator_ext4_high.v` | `instr[15:12]` を抽出し、16bitに符号拡張する。 | Level 1 |
| `imm_generator_ext8.v` | `instr[11:4]` を抽出し、16bitに符号拡張する。 | Level 1 |
| `imm_generator_ext12.v` | `instr[15:4]` を抽出し、16bitに符号拡張する。 | Level 1 |
| `imm_generator_mux4_16bit.v` | `mux2` を `generate` 文で組み上げ、4つの拡張結果から1つを選択する。 | Level 2 |
| `imm_generator.v` | 命令のOpcodeに基づき、適切な拡張形式を選択して出力するトップ。 | Level 3 |

### 3. パタン構造化と視覚化のポイント

#### 符号拡張の視覚化 (`ext` モジュール群)
符号拡張は単なる配線（`{{12{instr[7]}}, instr[7:4]}` 等）ですが、あえて独立したモジュールとして定義します。これにより、論理合成後の回路図で、**「命令から特定のビットが抜き出され、上位ビットに符号ビットがコピーされて太い16bitバスになる」**という流れが1つのブロックとして表示されます。

#### 命令形式と即値の対応
`imm_generator.v` において、Opcodeに応じてどの `ext` ブロックの出力を採用するかを決定します。
*   `addi`, `load`, `jalr` (Op: 1000, 1010, 1111) → `ext4_low`
*   `store` (Op: 1011) → `ext4_high`
*   `loadi`, `jal` (Op: 1001, 1110) → `ext8`
*   `blt`, `bz` (Op: 1100, 1101) → `ext12`

回路図上では、**「16bitの命令が4つの拡張器に同時に入力され、セレクタがそのうちの1つを拾い上げる」**という構造が見て取れます。これは「ハードウェアは並列に動いているが、選択によって意味が決まる」というデジタル回路の基本概念を教えるのに最適です。

#### 16bitセレクタの構造化
`imm_generator_mux4_16bit.v` は、1ビット単位のセレクタを `generate` 文で16個並べる構造にします。これにより、回路図上で「16bit幅の選択」が物理的な束として表現され、バスの概念を視覚的に補強します。
