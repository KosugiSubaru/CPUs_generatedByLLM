「ALUの入力ソース（レジスタ値 vs 即値）をどのように切り替えるか」を視覚化するため、`alu_src_selector`モジュールをビットごとに独立したマルチプレクサとして階層化・パターン構造化して提案します。

この構成により、論理合成後の回路図で、16ビットのデータバスがビットごとに並列にスイッチングされる様子が明確に表現されます。

### 1. 階層構造一覧（ファイル構成）

1ファイル1モジュールの原則に従い、以下の3つのファイルで構成します。

| レベル | ファイル名 | 役割 |
| :--- | :--- | :--- |
| **L0 (Top)** | `alu_src_selector.v` | 最上位モジュール。CPUデータパスのインターフェース。 |
| **L1 (Block)** | `alu_src_selector_mux_2to1_16bit.v` | 16bit幅の2入力マルチプレクサ。`generate`文で1bitセルを16個並列配置する。 |
| **L2 (Cell)** | `alu_src_selector_mux_2to1_1bit.v` | 選択の最小単位となる1bit 2入力マルチプレクサ。 |

---

### 2. 各モジュールの役割と階層レベルの詳細

#### L0: `alu_src_selector.v`
このモジュールは、ALUの「Operand B（第2入力）」の供給源を管理します。
- 命令デコーダからの制御信号（`i_alu_src_b_sel`）を受け取ります。
- 内部で `alu_src_selector_mux_2to1_16bit` をインスタンス化し、レジスタ出力（`rs2`）と即値生成器の出力（`imm`）を接続します。

#### L1: `alu_src_selector_mux_2to1_16bit.v` (パターン構造化)
16ビットという太い信号を、独立した1ビットのスイッチ16個の集まりとして定義します。
- **構造**: `generate`文を用いて、`alu_src_selector_mux_2to1_1bit` を16個生成します。
- **視覚的効果**: RTL Viewer等で確認した際、16個の同じ「箱」が整然と並び、それら全てに共通の「セレクト信号線」が一本貫いている様子が見えます。これはデジタル回路の「ビット並列処理」の美しさを教育的に示します。

#### L2: `alu_src_selector_mux_2to1_1bit.v`
選択ロジックの最小単位です。
- 2つの入力データ（d0, d1）と1つの選択信号（sel）から、1つのデータを出力します。
- **教育的意図**: 回路の最下層において、単一のビットがどのように制御信号によってゲートされているかを視覚化します。

---

### 3. モジュール呼び出しのイメージ（パタン構造化）

`alu_src_selector_mux_2to1_16bit.v` 内での実装イメージです。

```verilog
genvar i;
generate
    for (i = 0; i < 16; i = i + 1) begin : gen_alu_src_mux
        alu_src_selector_mux_2to1_1bit u_mux_bit (
            .i_sel (i_sel),  // 命令デコーダからの共通信号
            .i_d0  (i_data_rs2[i]),
            .i_d1  (i_data_imm[i]),
            .o_y   (o_alu_operand_b[i])
        );
    end
endgenerate
```

### 4. この階層化による教育的利点

1.  **「演算形式」のハードウェア的理解**: 
    - `add rd, rs1, rs2` (R-type) の時は `rs2` が選ばれる。
    - `addi rd, rs1, imm` (I-type) の時は `imm` が選ばれる。
    これらが単なる「プログラムの書き換え」ではなく、物理的な「配線の切り替え」によって実現されていることが一目でわかります。
2.  **制御信号の拡散の可視化**: 1本の制御信号（1bit）が、いかにして16ビット（16個のMUX）もの巨大なデータパスを支配しているかという、コントロールユニットの役割を物理的に提示できます。
3.  **モジュール再利用の概念**: 1bitのMUXという極小の部品を並べるだけで、16bitのデータセレクタができるという、ボトムアップ設計の考え方を視覚的に理解させることができます。
